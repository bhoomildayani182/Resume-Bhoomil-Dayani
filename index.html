Activity Logger

This project introduces a website using full stack development where backend technology was created using NodeJS, JavaScript API and to store information of subscribed use, MongoDB database. On running the project.

Our project basically generates a log file for any client to know how many times any activity has been used in his project and for what time it has been used. Operations engineers and developers use logs for debugging. Product managers and UX designers use logs for planning and design. Marketers want to track the performance of various features that relate to advertising campaigns.Node.js logging is an important part of supporting the complete application life cycle. From creation to debugging to planning new features, logs support us all the way. By analyzing the data in the logs, we can glean insights, resolve bugs much quicker, and detect problems early and as they happen.
